Medidas Absolutas

São elas: cm, mm, in(polegada), px, pt, pc

Medidas relativas

em, ex, rem, vw, vh, %

Importante!!

  • 1em = 16px
  • Recomendações W3C: em e px

Usando as fontes do Google Fonts

  1. Acessar fonts.google
  2. Escolher a fonte
  3. Selecionar os tipos das fonte que você escolheu
  4. Ir até "view select families" (Canto direito superior)
  5. Copiar o import e colar no css

Importando fontes externas (Não do Google Fonts)

  1. Baixar a fonte do site
  2. Adicionar um @font-face no css e colocar a url da fonte
